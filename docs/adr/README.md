# adr/ — Architecture Decision Records (the WHY)

One short file per non-obvious decision: what was chosen, the alternatives, and why. **Immutable** — a reversal is a NEW ADR that supersedes, not an edit.

- Name: `NNNN-short-title.md` (sequential: `0001-…`, `0002-…`).
- Shape: **Context** (the forces), **Decision** (what we chose), **Consequences** (trade-offs accepted), **Status** (accepted / superseded-by-NNNN).
- Write one when a choice is hard to reverse, surprising to a newcomer, or you'd otherwise re-litigate it next session.
- Not for: how-it-works (→ `reference/`), dated audits (→ `findings/`), or one-fact lessons (→ memory).
