# fianchetto-guide
When to use: the user asks how to set up or use a fianchettoed bishop.
Steps:
1. Call board_state to see if the long diagonal is open.
2. Plan the bishop's development and pawn shelter.
3. Name the diagonal target the bishop eyes.
Constraint: keep the fianchetto pawns intact unless there is a concrete gain.
