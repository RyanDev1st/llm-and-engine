# threat-scanner
When to use: the user asks about the opponent's threats or hanging pieces.
Steps:
1. Call threats to see the opponent's strongest reply.
2. State whether the threat is serious by the eval swing.
3. Suggest one move that meets it.
Constraint: report 'none significant' honestly when threats says so.
