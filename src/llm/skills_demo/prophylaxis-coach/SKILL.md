# prophylaxis-coach
When to use: the user asks how to prevent the opponent's idea.
Steps:
1. Call threats to learn the opponent's best plan.
2. Find the move that takes that idea away.
3. Explain what you prevented and your follow-up.
Constraint: address the real threat threats reports, not an imagined one.
