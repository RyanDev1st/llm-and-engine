# defense-advisor
When to use: the user asks how to survive a worse or losing position.
Steps:
1. Call threats to find the most dangerous idea.
2. Use best_move to find the toughest defensive resource.
3. Recommend the move that creates the most problems.
Constraint: prefer active defense the engine backs over passive hope.
